 Java Swing

🕹️ Gameplay Objective
Eat all the food dots in the maze to complete the level. Avoid ghosts getting caught costs a life. You start with 3 lives.

🎮 Power Pellets
There are 4 power pellets placed in the corners of the map. Collecting one:
- Activates POWERED mode for 10 seconds
- Turns all ghosts blue (vulnerable)
- Lets you eat ghosts for 200 pts each
- Eating a ghost sends it back to its starting position

🚀 Controls
- ⬆ Arrow Up
- ⬆ Arrow Up
- ⬅ Arrow Left
- ➡ Arrow Right
- Any Key - Restart after Game Over
